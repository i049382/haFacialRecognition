# Face Recognition Add-on

Home Assistant add-on for face recognition processing.

## Installation

Install via Home Assistant Add-on Store:

1. Add repository: `https://github.com/YOUR_USERNAME/haFacialRecognition`
2. Install "Face Recognition" add-on
3. Configure and start

See `GITHUB_SETUP.md` in the root directory for detailed instructions.
